Enter fasting glucose and insulin.

Enter both labs exactly as they appear in the chart. Use unit selectors to match the report—never convert manually.

Fasting glucose

Use the true fasting value (8+ hours).

Fasting insulin

Collected at the same time as glucose.

Formula

HOMA-IR = (glucose × insulin) / 405 when using mg/dL and µIU/mL.

Best practices

  • Both values from the same draw.
  • Patient fasted ≥ 8 hours.
  • Confirm units match the lab report.
