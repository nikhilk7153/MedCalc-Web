Checklist

Before calculating

Measured values

Copy both values exactly as shown on the lab report.

Guidance

Unit conversion

  • For mmol/L, the calculator handles the conversion automatically.
  • Consider ionized calcium if albumin is unreliable.

Before you submit

  1. Ensure both values are entered correctly.
  2. Check that units match your lab report.
  3. Return only the numeric corrected calcium.
